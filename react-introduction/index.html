What is React?

React is: A javascript library for building user interfaces.

To read more about react check out their website or just continue reading to jump right in.

React Components

React apps are made of components, which you can think of as representing a piece of the user interface. There's just one component here right now, the App component. You can see the output of that component under the section of this tutorial titled rendered react, at the bottom of this page. Head into the javascript section of this pen and try changing it.

Write your own component

Try writing your own component! First add the code below to the javascript section of this pen.

var HelloWorld = function() {
	return (
	  <p>Hello world!</p>
	);
}

Then add the component to the render function, it should look like this

ReactDOM.render(
	<HelloWorld />,
	document.getElementById('content')
);
